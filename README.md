Simon Says is a classic children's game where players follow commands given by a leader ("Simon") only when the phrase "Simon says" is used.

In this simplified version, the player is shown a sequence of symbols and must repeat it using a virtual or physical keyboard.

The player can use a hint once per round.

The goal is to correctly reproduce the sequence five times in a row to win.

There are three difficulty levels: easy, medium, and hard. The only difference between them is the set of symbols used.
